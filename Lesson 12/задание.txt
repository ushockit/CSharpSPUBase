Реализуйте консольное приложение "Библиотека"
За основу возьмите проект с урока
Ваша задача реализовать возможности:
- Поиск книг по названию
- Поиск книг по автору
- Поиск книг по жанру

Реализовать у каждой книги поле (в наличии(целое число))
Реализуйте возможность авторизации в системе. После авторизации, пользователь, имеет возможность взять книгу в прокат.
Взять книгу напрокат может только авторизованный пользователь.
У авторизованного пользователя появляется дополнительный пункт при просмотре книг "Взять напрокат" (если книга есть в наличии)
